
# Balkan sport


## Requirements

Node js (version 18) 

React js (version 18.2.0)

Mangodb 

Stripe Account


## Documentation

Download the project from GitHub, then install the node modules (both frontend and backend).


## Run Locally

Clone the project

```bash
  https://github.com/spaculussoft/test.git
```

Go to the backend directory

```bash
  cd test/backend
  
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Go to the frontend directory

```bash
  cd test/frontend
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start


```
